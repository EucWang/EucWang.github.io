kotlin的特征

  1. 野心很大, 替代java, 更简洁, 更高效, 更安全, 全平台支持

    • android
    • 服务器端
    • kotlin可以编译成Javascript
    • kotlin/原生 对c/c++/oc/swift的双端互操作

  2. 和java一样是静态类型的编程语言, 所有表达式的类型在编译期就确定了, 这个和Groovy,JRuby这些动态类型的语言是不同的. 虽然kotlin不用显示的声明变量类型.

  3. 函数式和面向对象

    • 头等函数, 函数可以作为值来使用, 可以作为参数传递,可以作为函数的返回值
    • 不可变对象
    • 纯函数.输入相同时返回相同的结果.
    • lambda表达式
    • 语法糖
      务实/简洁/安全/互操作行

  2. 带返回值的函数 和 语句表达式

    /**
* 返回值的类型 在参数列表后面声明, 以 ":"分开

* kotlin中没有三元表达式( a>b ? a : b)

fun max(a: Int, b: Int) : Int {
 return if (a > b) a else b    
}

目录和包

  1. kotlin文件中能以一条 package语句开头,声明文件中的都放到这个包中
  2. 两个文件可以声明成同一个包名, kotlin声明的包层级结构不需要和实际的文件目录结构相同,建议不用这样玩, 除非是特别小的类
  3. 不同的包名的声明, 使用时需要import导入, “.*” 导入全部
  4. kotlin的import导入不区分类和函数

使用when 处理枚举

 /**when可以是一个有返回值的表达式,代替java中的switch,break*/
fun getMnemonic(color: Color) =
    when(color) {
        Color.RED, Color.ORANGE -> "Of"   //"," 处理多个值合并到同一个分支
        Color.YELLOW -> "York"
        Color.GREEN -> "Gave"
        Color.BLUE -> "Battle"
        Color.INDIGO -> "In"
        Color.VIOLET -> "Vain"
    }

  1. 在when中使用任意对象, 比switch强大,switch要求必须是常量

    /**这里处理颜色的set集合, set中元素顺序不重要,只要包含一样的条目,就是相等的*/
fun mix(c1: Color,  c2 : Color)  =
 when(setOf(c1, c2)) {
     setOf(RED, YELLOW) -> ORANGE
     setOf(YELLOW, BLUE) -> GREEN
     setOf(BLUE, VIOLET) -> INDOGO
     else -> throw Exception("Dirty Color")  //结合else处理其他 的情况,这里抛异常
 }

  2. 使用不带参数的when,

    /**when的分支判断条件不仅仅是等式检查, 可以是任意的布尔表达式*/
fun mixOptimized(c1: Color, c2: Color) =
 when {
     (c1 == RED && c2 == YELLOW) ||
     (c1 == YELLOW && c2 == RED) -> ORANGE  //这里不会创建多个set集合

     (c1 == YELLOW && c2 == BLUE) ||
     (c1 == BLUE && c2 == YELLOW) -> GREEN

     (c1 == BLUE && c2 == VIOLET) ||
     (c1 == VIOLET && c2 == BLUE) -> INDIGO

     else -> throw Exception("Dirty color")
 }

类型检查和转换 (is, as)


interface Expr                                           //接口
class Num(val value: Int) : Expr                         //接口的实现类Num
class Sum(val left: Expr, val right: Expr) : Expr        //接口的实现类Sum

fun eval(e : Expr) : Int {
    if (e is Num) {               //is 类似于 java中的: instanceOf, 
        val n = e as Num     //强制转换了 类似于java中的: Num n = (Num)e
        return n.value
    } else if(e is Sum) {      //在类型判断成功的时候,kotlin就自动进行类型转换了,就不需要转换它了
        //☢注意,智能转换的对象的属性,必须是一个val属性,且不能有自定义访问器,防止发生变化
        return eval(e.right) + eval(e.left)  //递归调用
    }
    throw IllegalArgumentException("Unknown expression")
}

//对eval函数使用when重写(也可以用if来简化重写):
fun eval(e : Expr): Int = 
    when (e) {
        is Num -> e.value
        is Sum -> eval(e.right) + eval(e.left)
        else -> throw IllegalArgumentException("Unknown expression")
    }

kotlin中没有”java中的for循环”, 使用区间的概念替代

区间: 本质就是两个值之间的间隔, 两个值通常是数字: 一个起始值, 一个结束值, 使用 “..”运算符表示区间:
val oneToTen = 1..10
注意☢：kotlin区间是包含的或者是闭合的, 第二个值始终是区间的一部分

数列: 就是区间, 能够迭代区间中所有的值的区间

使用in运算符来检查一个值是否在区间中,
或者它的逆运算 !in 来检查一个值是否不在区间中

kotlin采用的还是java的集合类, 在java类的基础上进行了封装

/**按照格式输出一个集合中的全部元素*/
fun  joinToString(
    collection : Collection,       //需要显示的集合
    separator: String,    //分割符
    prefix : String,         //前缀字符串
    postfix: String)        //后缀字符串
    : String {                 //返回类型String

    val result = StringBuilder(prefix)               
    for((index, element) in collection.withIndex()) {   //带索引的遍历集合
        if(index > 0) result.append(seperator)            //添加分隔符到StringBuilder中
        result.append(element)                                 
    }
    result.append(postfix)
    return result.toString()   
}

joinToString函数的终极版本


fun  Collection.joinToString(
    separator: String=", ",
    prefix : String = "",
    postfix: String = ""
): String {
    val result = StringBuilder(prefix)
    for((index, element) in this.withIndex()) {
        if(index > 0) result.apend(separator)
        result.append(element)
    }
    result.append(postfix)
    return result
}

//java中的split()方法接受的字符串是正则表达式,kotlin对这个方法进行的重载

fun parsePath(path: String) {
    val dir = path.substringBeforeLast("/")           //几个String的扩展函数
    val fullName = path.substringAfterLast("/")

    val fileName = fullName.substringBeforeLast(".")
    val extension = fullName.substringAfterLast(".")
}

 //三重引号的字符串, 不需要对任何字符进行转义,包括反斜线
fun parsePath(path: String) {
    val regex  = """(.+)/(.+)\.(.+)""".toRegex()  
    val  matchResult = regex.matchEntire(path)
    if(matchResult != null) {
        val (dir, fileName, extension) = matchResult.destructured
        println("Dir: $dir, name: $fileName, ext: $extension")
    }
}
